About Me

Hi! My name is James. I am an English graduate who has just completed a foundation degree in Computing and IT with a focus in Computer Science.

I really enjoyed the coding aspects of my studies and learning how computers function. As I have just finished, I am intending to take a year or two out of studying and get a job in the IT industry, preferably web development. Then, if I feel able to and still wish to, I will return to top-up my Diploma of Higher Education into a Bachelor of Science whilst still working.

My current developer skills include:

  • Web development languages: HTML, CSS, and JavaScript
  • Building websites using: React and Node.js
  • Setting up databases with: PHP and SQL in MySQL and MongoDB
  • Creating programs in: Python and Java

My ultimate goal is to become a full-stack web developer as I am passionate about creating intuitive and efficient web applications that solve real-world problems. In the next few years, I aim to gain hands-on experience in the industry, working on diverse projects that challenge my skills and expand my knowledge.

Additionally, I am interested in exploring other career paths within IT, such as programming and software engineering, as I also find these fields fascinating and align with my interests in coding. I could see myself possibly transitioning into Software Engineering over time.

SnakeGame Project

This SnakeGame project is a Java-based remake of Nokia's classic Snake game.

Developed as a personal project, it was a great opportunity to practice and enhance my coding skills while working on something fun and nostalgic.

Project Highlights:

  • Language: Java
  • Custom Sprites: The game retains the general rules and gameplay of the original Snake game but features custom-designed sprites for a unique visual experience.
  • Objective: Guide the snake to eat food, growing longer with each piece consumed, while avoiding collisions with the walls and the snake's own body.

This project was not only enjoyable to create but also a valuable learning experience, solidifying my understanding of Java and game development principles.

Currency Converter Project

This Currency Converter project is a Python-based application designed to convert values between different currencies.

Developed as a personal project, it was an excellent way to practice coding and understand financial calculations and exchange rates.

Project Highlights:

  • Language: Python
  • Functionality: The application allows users to input an amount in one currency and convert it to another using up-to-date exchange rates.
  • User Interface: Features a simple and intuitive GUI for easy interaction and smooth user experience, built using Tkinter.
  • Real-time Exchange Rates: Integrates with an API to fetch current exchange rates, ensuring accurate and reliable conversions.

Creating this project was a rewarding experience, allowing me to apply my Python skills to a practical tool that can be useful in real-world scenarios.
